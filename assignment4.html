Q4.Learn <label> for accessibility and placeholder for hints. Connect labels with inputs using for and id.

What is < label >?
The <label> tag provides a description for an input field. It improves accessibility because users can click the label to focus on the input.
What is placeholder?
The placeholder attribute displays a hint inside the input box before the user enters data.
